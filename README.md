# Portfolio-Optimization-using-Excel

This project is used to optimize a portfolio by reducing risk using Simplex Method

## Tools Used:
- Microsoft Excel for assigning weights and risk calculation
- Google Sheets and Google Sheets Extensions for getting price of Stocks of 3 years

## What I learned:
- Calculate the deviation of the stock price from its average
- Understand how simplex method is utilised in real world scenario\
- How to create a complex model in Excel
